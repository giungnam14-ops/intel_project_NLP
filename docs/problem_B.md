# 문제 후보 [B]

## 누구의 문제인가 (대상 사용자)
- 앱, 웹사이트, 쇼핑몰, 구독 서비스 이용 전에 약관이나 동의서를 확인해야 하는 사용자
- 무료체험이나 정기결제를 이용하면서 자동결제, 환불, 해지 조건을 놓치기 쉬운 사용자
- 개인정보 제공이나 마케팅 수신 동의 내용을 빠르게 확인하고 싶은 사용자

## 어떤 상황에서 겪는가
- 앱 회원가입 과정에서 긴 이용약관과 개인정보 처리방침에 동의해야 할 때
- 무료체험 신청 후 자동결제가 되는지 확인해야 할 때
- 구독 서비스 해지 조건이나 환불 가능 여부를 확인해야 할 때
- 쇼핑몰, 멤버십, 이벤트 참여 과정에서 개인정보 제공 범위를 확인해야 할 때

## 반복적으로 겪는 페인 포인트 (관찰/인터뷰 근거)
- 약관과 동의서가 너무 길어 대부분 자세히 읽지 않고 동의한다.
- 자동결제, 환불 제한, 해지 조건 같은 중요한 내용이 작은 문장 속에 숨어 있다.
- 개인정보가 어디까지 제공되는지 바로 파악하기 어렵다.
- 사용자는 약관을 읽지 않고 넘긴 뒤 나중에 결제나 해지 문제를 겪을 수 있다.
- 중요한 조건을 찾기 위해 직접 검색하거나 여러 번 다시 읽어야 한다.

## 지금은 어떻게 (불편하게) 해결하고 있나
- 약관을 대충 훑어보고 동의한다.
- 자동결제나 환불 관련 단어를 Ctrl+F로 직접 검색한다.
- 결제 후 문제가 생기면 고객센터나 FAQ를 찾아본다.
- 이해하기 어려운 문장은 검색하거나 AI에게 따로 물어본다.
- 대부분은 귀찮아서 확인하지 않고 넘어간다.

## 이 문제가 중요한 이유
- 약관과 동의서에는 사용자의 결제, 환불, 개인정보, 해지 조건과 관련된 중요한 내용이 포함되어 있다.
- 사용자가 내용을 제대로 확인하지 못하면 원치 않는 자동결제, 환불 불가, 개인정보 제공 같은 불편을 겪을 수 있다.
- 기존 약관 요약은 내용을 줄여줄 수는 있지만, 사용자가 동의 전에 반드시 확인해야 하는 주의 문장을 한눈에 보여주지는 못한다.
- 따라서 약관 속 주의 문장을 자동으로 탐지하고 쉬운 말로 보여주는 서비스가 필요하다.
